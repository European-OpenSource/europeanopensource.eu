Distrobox is a shell tool by Italian developer Luca Di Maio that runs any Linux distribution inside a container in the terminal, with full host integration. Using Podman, Docker, or lilipod, it shares the home directory, display, audio, and USB devices — enabling seamless cross-distro software compatibility on any Linux system.
